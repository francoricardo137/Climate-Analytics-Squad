# Climate Analytics Squad

## ¿Qué es esto?
Trabajo práctico para Organización Empresarial (UTN). Analizamos datos climáticos para practicar Git, GitHub y Jira.

## Integrantes

## Escenario elegido
Escenario A - Datos climáticos (temperaturas)

## Dataset
Usé datos de temperatura global de datahub.io/core/global-temp

## Estructura del proyecto
- /datos -> acá van los CSV
- /scripts -> acá está el script de Python
- /resultados -> acá se guardan los gráficos y el resumen

## Cómo correrlo
1. Poner el archivo CSV en la carpeta /datos
2. Ejecutar: python scripts/analisis_climatico.py
3. Los resultados aparecen en /resultados
